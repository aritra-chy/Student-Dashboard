# Student Dashboard

This is a simple Student Dashboard application built using React and Vite. The project allows users to view, add, and delete student information. It was developed to practice basic React concepts such as components, props, state management, forms, event handling, conditional rendering, and error boundaries.

## Features

- View a list of students
- Display total number of students
- Add new students using a form
- Delete students from the list
- Show student status (Active/Inactive)
- Conditional rendering using different React techniques
- Error Boundary for handling application errors

## Technologies Used

- React
- Vite
- JavaScript
- CSS

## Installation

1. Clone the repository:

```bash
git clone <repository-link>
```

2. Go to the project folder:

```bash
cd student-dashboard
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```
